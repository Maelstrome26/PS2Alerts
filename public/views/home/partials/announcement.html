Homepage Statistics have been expanded!

Victory Timeline

This handy graph shows the victories for each faction over time, allowing you to see the trends that are occuring. This graph will be improved over time, with labels detailing when major patches landed so you can see their affect on alerts coming soon.

Victory Breakdowns

There's now 4 new statistics panels which show you how each server's victories are distributed. E.g. You're now able to see which servers the Vanu Soverignity are doing best on Indar. (As of writing, that's Miller and Cobalt!).

Global Server Victories - Victories broken down by each server, showing how each faction are doing on each server.

Global Continent Victories - Victories broken down by continent, showing which faction is dominating Indar! (As of writing, that's the TR!)

Per-Server Continental Victories - Victories broken down by Server and by Continent, which allows you see how each continent fairs for each faction on each server.

Faction Victory Contributions - Shows a breakdown of the total number of wins, per faction, and how they contribute to the overall faction percentages. Shown in both a segmented bar chart and in tabular form. Interested to see which servers are pulling their weight? Now you can find out!
