PS2Alerts will be back fully soon!

PS2Alerts' webserver got hacked recently, which unfortunately meant that the website had to be taken down. It is currently under reconstruction, an early version of which you can see below.


This is a very basic version! There are lots of features missing and they will return!


The new version of the website will also feature a complete redesign, as you can see by the very simple example that's shown below. I've learnt a lot about coding since I did PS2Alerts v4.0 back in October last year, so I think it's time to apply said knowledge.


In bringing the webserver back online (in a clustered based setup) I decided it was prudent that I install PHP7, the latest and greatest in server side scripting, which makes any PHP site at least twice as quick.


Unfortunately this meant, that due to the method of coding I was using for the old PS2Alerts website, that it would no longer function and was not compatible. This meant that the website was left crippled until I could have the time to fix it.


However, the collection script has still been running, at least when I brought that particular server back. This means while I rewrite the website, that the statistics will still be collected in the background.


So here we are. A new version of PS2Alerts is in the making. Rather than having a dead website, I decided that having a very basic version of the upcoming site would be prudent.
