Change Log

See whats changed recently!

24th April 2016

Player Profiles - I'm happy to present the first iteration of Player Profiles! With player profiles, you'll be able to see the statistics for any particular player we have on record, including stats such as:

  • Kills, Deaths, Teamkills, Suicides and headshots across all alerts recorded
  • Averages, such as K/D, Kills & Deaths per alert etc
  • A list of all alerts that a player has participated in
  • All weapon kills, headshots and teamkills a player has accumulated
  • All vehicle metrics - note, that this metric is a bit dodgy and may be removed

3rd March 2016

Leaderboards have been implemented on alert pages! You're now able to see what you ranked in each and every alert! There are currently 4 leaderboards: Players, Outfits, Weapons and Vehicles.


Project status widgets have been added to the site, detailing the status of the current page you're looking on, keeping you up to date with the progress of the project as well as letting you know where to post suggestions.

27th February 2016

The initial implementations of the Alert History and Alert Detail pages have been launched! The Alert History page allows you to see all alerts in the past 48 hours (a temperary measure until a better date selection system can be made) and you're also able to see basic information and leaderboards on a per-alert basis.

The new version of the website will also feature a complete redesign, as you can see by the very simple example that's shown below. I've learnt a lot about coding since I did PS2Alerts v4.0 back in October last year, so I think it's time to apply said knowledge.
